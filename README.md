## Hyprland Setup

To install and set up the Hyprland dotfiles, follow these steps:

<small> 

> **Note:** The hyprland.sh works better using a minimal install of Arch Linux using the archinstall script.

</small>

1. Clone the repository:

    ```sh
    git clone https://github.com/seethastars/dotfiles.git
    ```

2. Change to the repository directory:

    ```sh
    cd dotfiles/
    ```
    
3. Run the install script:

    ```sh
    ./hyprland.sh
    ```
   
4. Enjoy Hyprland!

## Keybinds

<div align="center">

| Mapping                 | Action                                           |
| ----------------------- | ------------------------------------------------ |
| `Super + Return`        | Open Terminal                                    |
| `Super + Q`             | Close Window                                     |
| `Super + W`             | Open Web Browser - (Librewolf)                   |
| `Super + E`             | Open File Manager - (Thunar)                     |
| `Super + R`             | Open App Launcher - (Wofi)                       |
| `Super + L`             | Exec lockscreen - (Hyprlock)                     |
| `Super + B`             | Restart Waybar                                   |
| `Super + N`             | Change wallpaper                                 |
| `Super + M`             | Exit Hyprland                                    |
